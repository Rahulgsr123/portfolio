Projects

Netflix UI Showcase

• This Netflix inspired web app is a responsive streaming interface built using HTML, CSS, JavaScript, React.js, Firebase, and Bootstrap. Its purpose is to replicate core functionalities of Netflix, offering users a curated UI for browsing and viewing media content.
• Key features include user authentication via Firebase, dynamic routing with React Router, and interactive browsing through genre-specific movie files using TMDB API. Components were modularly built to support scalability and fast rendering.
• UI optimization was handled through Bootstrap’s grid system and custom CSS tweaks for responsiveness across devices. Lazy loading techniques were implemented to reduce initial page load times and enhance perceived performance.
• Focused enhancements were made to the homepage layout and login interface for aesthetic appeal and intuitive navigation. The colour palette and typography were fine-tuned to reflect Netflix's branding while ensuring accessibility.
